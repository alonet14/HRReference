Installation
=============

To use the module, please put the radarsimpy folder within your project folder as shown below.

**Windows**

::

    - your_project.py
    - your_project.ipynb
    - radarsimpy
        - __init__.py
        - radarsimc.dll
        - scene.xxx.pyd
        - ...


**Linux**

::

    - your_project.py
    - your_project.ipynb
    - radarsimpy
        - __init__.py
        - libradarsimc.so
        - scene.xxx.so
        - ...
